# Skill Content

- **Commands**: Preferred use of `docker buildx` and `docker compose`.
- **Optimization**: Multi-stage builds for smaller images.
- **Security**: Run as non-root user where possible.
- **Persistence**: Explicit volume mapping for stateful applications.
