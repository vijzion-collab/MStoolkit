# Skill Content

- **Style**: Follow "Effective Go" and Uber Go Style Guide.
- **Error Handling**: Explicitly check errors; do not use panics for flow control.
- **Concurrency**: Use goroutines and channels sparingly and correctly.
- **Testing**: Include unit tests with `testing` package for all new logic.
