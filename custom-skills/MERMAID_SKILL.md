# Skill Content

- **Tooling**: Use Mermaid.js syntax for all diagrams.
- **Types**: Support Flowcharts, Sequence Diagrams, Gantt charts, and Entity Relationship diagrams.
- **Integration**: Embed diagrams directly in markdown artifacts using the `mermaid` code block identifier.
- **Best Practice**: Keep labels concise and use subgraphs to group related components.
